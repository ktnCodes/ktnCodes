# Workspace

Kevin's professional portfolio and career development hub. Single source of truth for code projects, the persona layer (`00-agentic-OS`), the public site, and the knowledge-vault.

The naming convention is position-addressed: folder = type, prefix = priority. New work earns a folder slot. Old work doesn't get to take up visual real estate.

Rules and identity for all AI agents live in `AGENTS.md` (canonical). `CLAUDE.md` and `copilot-instructions.md` are 1-line shims pointing back to it.
