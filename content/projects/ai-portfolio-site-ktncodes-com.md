# AI Portfolio Site (ktncodes.com)

Production Next.js 15 portfolio with an agentic chatbot using Vercel AI SDK. Implements tool-use chaining (up to 3 steps per turn) with tools for resume, projects, skills, contact, blog search, and on-demand MDX post content retrieval. Multi-model fallback (Gemini → OpenAI) with in-flight retry logic and server-side quota tracking.
