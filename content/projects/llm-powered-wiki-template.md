# LLM-Powered Wiki Template

Internal hackathon project. Forkable team wiki template built on Karpathy's 3-layer architecture (raw -> compiled -> operating instructions), LLM-maintained through agent skills. Ships with 6 pipeline skills for setup, ingest, query, lint, deploy, and repo self-mapping. ICM provides self-documenting context so any LLM can orient instantly in a fresh fork.
